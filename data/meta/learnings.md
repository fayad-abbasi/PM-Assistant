# Learnings & Corrections

This file captures mistakes Claude makes and corrections from the user.
It is injected into every session via the session-start hook so the same mistake never happens twice.

## How to Use

When Claude produces an output that needs correction, add the learning here in this format:

```
### [Date] — [Context]
**Mistake**: What Claude did wrong
**Correction**: What to do instead
**Why**: Why this matters
```
